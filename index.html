Features

SkyCalc blends spreadsheet modeling and general programming in a coherent and intuitive framework. The result is a powerful productivity app with a lot to offer to different users:

For begineers
SkyCalc is a great way to build spreadsheets, while acquiring basic programming skills. Novice users quickly learn to do things like:
- Create worksheets, edit formulas, add/remove rows and columns, etc.
- Reuse spreadsheet templates to build non-trivial models
- Drag and drop worksheets and other components from a palette of variable widgets
- Organize worksheets and other components in different tab panels, for better visualization
For intermediate users
SkyCalc offers powerful mechanisms to perform tasks like:
- Define complex data structures, with fields that can hold data as well as formulas
- Create queries to perform sophisticated analysis on arbitrary data
- Define new functions and create reusable components
For advanced users
SkyCalc offering to expert users shines in comparison to traditional spreasheets:
- Higher order functional programming
- Data abstraction: users can define new types along with their operations
- Monte Carlo simulations powered by various stochastic distributions
- Interprocess communication: channels allow SkyCalc instances to communicate with each other
- High Performance Computing: multiple SkyCalc instances can be deployed on a cluster

FAQ

Do we need yet another web-based spreadsheet?
Yes! Because SkyCalc is much more than a spreadsheet. SkyCalc is a complete programming environment in the cloud. Nothing combines spreadsheet modeling and general computing as well as SkyCals does.
Is SkyCalc offered only as a web-based product?
No. SkyCalc can also be licensed for private clouds as well as a standalone product.
When will SkyCalc be ready?
We are targeting Q4 2014 for our first release. Currently there are a handful of Beta customers using our bare bones API, with no GUI.
